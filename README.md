# Attendance-Management-System-Java-
The Attendance Management System is a Java-based desktop application developed using Java Swing and File Handling. It allows users to add students, mark daily attendance, store records in a file, and calculate attendance percentages. This project demonstrates concepts of GUI programming, event handling, collections, and persistent storage.
